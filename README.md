# address-quality-detector
An NLP-based machine learning application trained on Indian addresses to detect invalid or low-quality addresses. Built to enhance delivery success rates, reduce operational returns, and improve overall data quality in e-commerce and logistics systems.

# Live Demo
https://huggingface.co/spaces/swatigupta110/space-address-quality-detector

# Features
- Detects invalid addresses
- Classifies address quality using a trained machine learning model
- Simple web interface for testing addresses
- Deployed online for live testing

## Framework
- Gradio: used to build the web interface for the model and run it on Hugging Face Space.
## Machine Learning / NLP Libraries
- FastText: used for training/loading the text classification model.
- scikit-learn: used for preprocessing, evaluation, or utilities.
## Data Processing
- Pandas – used for handling the dataset.
## Deployment Platform
- Hugging Face Spaces – used to deploy and host the live demo.
## Programming Language
- Python
